# Etch-a-Sketch


## Description
A browser-based drawing tool built with vanilla JavaScript. 
Hover over the grid to draw with random colors that progressively darken.

## How to use
- Hover over the grid squares to draw
- Each square darkens by 10% on each interaction
- Click "Nouvelle Grille" to create a new grid of custom size (max 100x100)

## Technologies
- HTML
- CSS
- JavaScript
